# Installation

Add the libraries located in the ProQuest\lib folder to the Referenced Libraries in your project. 

Java Version: java 1.8.0_192
Selenium Version: selenium-java-3.141.59
Chrome Version: Version 79.0.3945.79 (64-bit)

Libraries to be Added as Referenced Libraries:
/ProQuest/lib/byte-buddy-1.8.15.jar
/ProQuest/lib/client-combined-3.141.59-sources.jar
/ProQuest/lib/client-combined-3.141.59.jar
/ProQuest/lib/commons-exec-1.3.jar
/ProQuest/lib/commons-io-2.6.jar
/ProQuest/lib/guava-25.0-jre.jar
/ProQuest/lib/jsoup-1.12.1.jar
/ProQuest/lib/okhttp-3.11.0.jar
/ProQuest/lib/okio-1.14.0.jar

Chrome Driver Location: ProQuest\drivers\chromedriver.exe

# Usage

Run ProQuest\src\GoogleSearch.java 
Run ProQuest\src\Screenshot.java

Default Output Location: ProQuest\test-results
